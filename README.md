# Barber Luxury - Frontend

Este es el frontend de la aplicación **Barber Luxury**, una plataforma para reservas en barberías que permite a los clientes agendar sus turnos con barberos disponibles, actualmente **Juan** o **Camilo**.

## 🚀 Sitio en producción

Accede a la aplicación aquí:  
👉 [https://barberluxury.netlify.app/](https://barberluxury.netlify.app/)

## 🧾 Descripción

La aplicación permite a los clientes:

- Ver la disponibilidad de los barberos.
- Reservar sus turnos de manera sencilla.
- Navegar por una interfaz limpia y amigable.

## 🛠️ Tecnologías utilizadas

- **React** – Framework de JavaScript para la construcción de interfaces de usuario.
- **Tailwind CSS** – Framework de utilidades para estilos rápidos y responsivos.
- **Axios** – Cliente HTTP para la comunicación con el backend.
- **React Router DOM** – Para la navegación entre rutas en la SPA.
